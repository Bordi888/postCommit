HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents form a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and othe objects, such as interactive forms, may be embedded into thhe rendered page. it provides a means to create structured documents by denoting structural semantics for text such as headings paragraphs, lists, links, quotes and other items. html elements are delineated by tags, written usiing angle brackets.
